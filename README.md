# DIGIVORA – Portofolio Kreatif Digital

DIGIVORA adalah website portofolio kreatif digital yang dibuat sebagai proyek tugas kelompok.  
Website ini menampilkan identitas tim, karya, serta artikel insight yang berkaitan dengan dunia kreatif dan digital.

## Tujuan Project

- Menyajikan portofolio digital dalam bentuk website yang modern dan responsif
- Melatih kerja tim dan kolaborasi menggunakan GitHub
- Mengimplementasikan HTML, CSS, dan JavaScript sesuai standar web

## Fitur Utama

- Halaman Home ( Landing Page)
- About Me (Profil Singkat)
- Blog / Artikel
- Project / Portofolio
- Contact / Get In Touch
- Desain responsif (mobile friendly)
- Dark/Light Mode Toggle
- Interactive Project Gallery
- Smooth Navigation & Scroll

## Teknologi yang Digunakan

- HTML5 (Semantic Markup)
- CSS3 (Flexbox & Responsive Design)
- JavaScript
- Font Awesome
- Git & GitHub
- Visual Studio Code
